Net Salary Calculator - README

This program calculates an individual's net salary by getting the inputs of basic salary and benefits. It calculates the following:

  • Payee (Tax)
  • NHIF Deductions
  • NSSF Deductions
  • Gross Salary
  • Net Salary

Usage:

  1. Enter the individual's basic salary when prompted.
  2. Enter any additional benefits (if none, enter 0) when prompted.
  3. The program will calculate and display the gross salary, tax, NHIF deductions, NSSF deductions, and net salary.

Example:

Enter your basic salary: 50000

Enter any additional benefits (if none, enter 0): 10000

Gross Salary: $60,000.00

Payee (Tax): $18,000.00

NHIF Deductions: $1,200.00

NSSF Deductions: $3,000.00

Net Salary: $37,800.00
