Student Grade Calculator - README

This program calculates and assigns grades based on the following criteria:

  • A: Marks greater than 79
  • B: Marks between 60 and 79
  • C: Marks between 50 and 59
  • D: Marks between 40 and 49
  • E: Marks less than 40

Usage:

  1. Run the program.
  2. Enter the student's marks when prompted (should be between 0 and 100).
  3. The program will display the corresponding grade based on the input marks.

Example:

Enter student marks (0-100): 85

Grade: A

Enjoy using the Student Grade Calculator!
